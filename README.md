# Three Men's Morris Game

A React implementation of the classic Three Men's Morris board game.

## Game Rules

Three Men's Morris is a strategy board game played on a special board with designated positions. The game is played between two players with the following rules:

- Each player starts with 6 pieces
- Players take turns placing their pieces on valid positions on the board
- When a player forms a line of three pieces (horizontally, vertically, or diagonally), they score a point and remove one of their opponent's unplayed pieces
- The game ends when either player runs out of pieces
- The player with the highest score wins

## Features

- Interactive game board
- Score tracking
- Turn-based gameplay
- Valid move highlighting
- Game over detection
- Responsive design

## Technologies Used

- React

## Setup and Installation

1. Clone the repository:
```bash
git clone https://github.com/ceSamet/three-mens-morris.git
```

2. Navigate to the project directory:
```bash
cd three-mens-morris
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm start
```

5. Open [http://localhost:3000](http://localhost:3000) to play the game in your browser.

## Development

To build the project for production:
```bash
npm run build
```
